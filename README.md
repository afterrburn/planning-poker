# Planning Poker - Hallucinators Team

A simple real-time planning poker app for sprint estimation.

## Live App

**https://afterrburn.github.io/planning-poker/**

## Features

- Real-time voting with Fibonacci points (0, 1, 2, 3, 5, 8, 13, 21, ?, â˜•)
- Share room links with your team
- Reveal votes when everyone is ready
- Reset for new rounds
- Create new games with auto-generated URLs

## Usage

1. Open the app URL
2. Enter your name
3. Enter a room name (or leave blank for auto-generated)
4. Share the room link with your team
5. Vote, reveal, repeat!
